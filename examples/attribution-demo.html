4. PHP Code Examples

How to generate these attributions in PHP:

Basic Attribution:

$client = TVMazeClient::create('MyApp/1.0');
echo $client->getAttributionHtml();
// Output: <p class="tvmaze-attribution">Data provided by <a href="https://www.tvmaze.com" target="_blank" rel="noopener noreferrer">TVMaze</a></p>

Detailed Attribution:

echo $client->getDetailedAttributionHtml();
// Includes CC BY-SA 4.0 license information

Model-specific Attribution:

$show = $client->getShow(1);
echo $show->getAttributionHtml();

$episode = $client->getShowEpisodes(1)[0];
echo $episode->getAttributionHtml();

Different Formats:

// HTML for web
echo $client->getAttributionHtml();

// Plain text for console
echo $client->getAttributionText();

// Markdown for documentation
echo $client->getAttributionMarkdown();

5. License Compliance

Important: The TVMaze API is licensed under Creative Commons Attribution-ShareAlike 4.0.

This means you must provide attribution when using their data. The attribution methods in this client make it easy to comply with these requirements.

Requirements:

  • ✅ Attribution - Credit TVMaze as the data source
  • ✅ ShareAlike - Share modifications under the same license
  • ✅ Link - Provide a link to TVMaze

6. Security Features

Notice the rel="noopener noreferrer" attributes in the links:

  • noopener - Prevents the new page from accessing window.opener
  • noreferrer - Doesn't send referrer information
  • target="_blank" - Opens in new tab/window

These attributes ensure security and privacy when linking to external sites.
